# 典型测试 prompt

## 正常请求

```text
$via-ppt-rescue-maker 我下周要做 10 页课程汇报，主题是“大学生如何用 AI 提升学习效率”。请帮我从 0 到 1 完善内容逻辑，并用 image 模型生成每页 PPT 图片。
```

## 有粗略大纲

```text
$via-ppt-rescue-maker 我有个大纲：背景、问题、案例、启发、建议。主题是“县城咖啡店为什么越来越多”，做 8 页课堂展示。请帮我把这个大纲变成能讲的 PPT，并生成页面 PNG。
```

## 有完整文档

```text
$via-ppt-rescue-maker 我有一篇完整调研文档，主题是“短视频使用对大学生注意力的影响”，想改成 12 页开题汇报。请重组研究问题、文献缺口、方法和可行性，先定内容蓝图，再生成每页 PPT 图片。
```

## 有散乱素材

```text
$via-ppt-rescue-maker 我手里有一些用户评论截图、访谈摘录和自己的想法，想做一份 10 页产品复盘，主题是“AI 写作工具为什么新用户第二天就不用了”。请帮我整理成能汇报的 PPT 页面 PNG。
```

## 有旧 PPT

```text
$via-ppt-rescue-maker 我有一份旧 PPT，主题是“社区养老服务数字化”，现在逻辑很乱、页面也丑。请帮我先诊断，再重做成 10 页答辩风格汇报，并生成每页图片。
```

## 有数据表

```text
$via-ppt-rescue-maker 我有一张小红书账号近 30 天数据表，想做 8 页内容复盘。请帮我找核心发现、解释原因、给下周选题建议，并生成每页 PPT 图片。
```

## 边缘测试

```text
$via-ppt-rescue-maker 我明天组会要讲一个神经科学主题：运动如何影响海马神经发生和情绪调节。请做 12 页机制综述型组会 PPT，要有理论模型、实验范式、变量设计、数据分析框架、关键证据、局限和讨论问题。先把 PPT 里到底讲什么定死，再用 image 模型生成每页图片。
```

## 首图确认测试

```text
$via-ppt-rescue-maker 我想做 8 页职场午餐和下午精力管理的内训 PPT，听众是互联网公司员工。请先确认页序，页序确认后只生成两张风格确认样张：第 1 页和一张最有代表性的内容页，不要在我确认风格前批量生成剩余图片。
```

## 不应触发的请求

```text
帮我把一段文字改得更口语化，不需要 PPT。
```
